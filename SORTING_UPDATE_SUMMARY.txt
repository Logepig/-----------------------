✅ Сортировка по дате создания (created_at)
✅ Работает для таблиц: users и projects
✅ Три режима сортировки:
   • 1 клик → Новые сверху (DESC) ↓
   • 2 клика → Старые сверху (ASC) ↑
   • 3 клика → Исходный порядок

✅ Визуальные индикаторы (стрелки ↑↓)
✅ Подсветка активной сортировки
✅ Плавные анимации
✅ Адаптивный дизайн

═══════════════════════════════════════════════════════════════════
  КАК ИСПОЛЬЗОВАТЬ
═══════════════════════════════════════════════════════════════════

1. Откройте админ-панель (admin/admin123)
2. Выберите таблицу "users" или "projects"
3. Кликните на заголовок колонки "created_at"
4. Наблюдайте изменение порядка записей
5. Кликните еще раз для смены направления
6. Кликните третий раз для отмены

═══════════════════════════════════════════════════════════════════
  ТЕХНИЧЕСКИЕ ДЕТАЛИ
═══════════════════════════════════════════════════════════════════

Алгоритм:
  • Сортировка на клиенте (быстро)
  • Исходные данные кешируются
  • Создается копия при сортировке
  • Минимум DOM операций

Производительность:
  • Отлично для < 1000 записей
  • Хорошо для < 10000 записей
  • Может быть медленно для > 10000 записей

Состояние сортировки:
  {
    column: 'created_at',
    order: 'desc' | 'asc' | null,
    originalData: [...]
  }

  ВИЗУАЛЬНЫЕ ИНДИКАТОРЫ
═══════════════════════════════════════════════════════════════════

Заголовок created_at:
  • Обычное состояние: Стандартный текст
  • Hover: Подсветка синим (var(--primary))
  • Активная сортировка DESC: created_at ↓
  • Активная сортировка ASC: created_at ↑
  • Отмена: Без индикатора

Цвета:
  • Активный: #1e88e5 (light) / #60a5fa (dark)
  • Фон: var(--bg-elev)
